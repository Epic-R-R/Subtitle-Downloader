## Getting Started
To start you need to install python >= 3.5 and pip
### Prerequisites

* python
```sh
sudo apt install python3
```
* pip
```sh
sudo apt install python3-pip 
```


### Installation

1. Clone the repo
```sh
git clone https://github.com/Epic-R-R/Subtitle-Downloader.git
```
2. Go to directory 
```sh
cd SsubDownloader
```
3. Install Tool
```sh
sudo python main.py
```
4. Run SsubDownloader
```sh
ssubdownloader
```
